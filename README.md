# 一款精致的Flutter版 Wanandroid App.
## 旨在打造一款完整的Flutter应用项目。  
包含启动页，引导页，主题色切换，应用国际化多语言，版本更新等功能，项目主框架采用响应式编程RxDart和BLoC，由于目前项目功能尚未完善，代码还有待优化，后续项目重构整理好后，会全部开源出来，敬请期待！欢迎下载体验。  

有关项目最新动态，可以关注App内第一条Hot Item信息。
